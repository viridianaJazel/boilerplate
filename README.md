# Boilerplate

Boilerplate for a react application with redux, saas and jest

### Installation and execution

#### Development environment
```
$ npm install
$ npm run dev
```

#### Production environment
```
$ npm install
$ npm run build
$ npm start
```

#### Run eslint

```
$ npm run lint
```

#### Run unit testing

```
$ npm run test
```

To run code coverage reports use the following command.

```
$ npm run cover
```
